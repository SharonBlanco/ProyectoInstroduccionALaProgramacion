# README

## Proyectos del primer semestre – Ingeniería en Computación (TEC)

Estos proyectos fueron desarrollados durante mi **primer semestre en la carrera de Ingeniería en Computación en el Instituto Tecnológico de Costa Rica (TEC)**, en el contexto de los cursos de **Introducción a la Programación y Taller de Programación**.

El objetivo principal de estos proyectos fue **aplicar los fundamentos de programación en Python**, estructuras de datos y técnicas básicas de análisis de información a un problema práctico.

## Descripción del proyecto

El proyecto consiste en una aplicación sencilla que funciona como un **entrevistador asistido por inteligencia artificial**, capaz de:

- Capturar video desde una cámara web.
- Detectar rostros en los fotogramas utilizando **Haar Cascades de OpenCV**.
- Analizar expresiones faciales mediante un **modelo de Deep Learning** para identificar emociones.
- Almacenar información personal del entrevistado utilizando **listas y diccionarios en Python**.
- Registrar las emociones detectadas en cada fotograma durante la entrevista.
- Realizar consultas sobre los datos recolectados utilizando distintos algoritmos de ordenamiento.

Las entrevistas consisten en que el entrevistado relate experiencias relacionadas con diferentes emociones (sorpresa, tristeza, enojo y felicidad), mientras el sistema registra los datos y analiza sus expresiones faciales.

## Tecnologías utilizadas

- NumPy

## Nota sobre el modelo de inteligencia artificial

El **modelo de reconocimiento de emociones y parte del código de captura de video** fueron proporcionados por el profesor del curso como base para el proyecto.

El enfoque principal del trabajo fue **desarrollar la lógica del programa, la estructura de datos y las consultas sobre la información recolectada**, aplicando los conceptos aprendidos en clase.

## Funcionalidades implementadas

El sistema permite realizar varias consultas sobre las entrevistas almacenadas:

1. Mostrar toda la información recolectada de las personas entrevistadas.
2. Ordenar entrevistas por edad utilizando **Insertion Sort**.
3. Ordenar entrevistas por cantidad de hijos utilizando **Bubble Sort**.
4. Ordenar entrevistas por cantidad de hijos utilizando **QuickSort**.
5. Generar listas de emociones detectadas en cada entrevista.
6. Ordenar emociones según el nivel de felicidad detectado.
7. Generar gráficos comparativos de emociones durante la entrevista.

## Propósito académico

Este proyecto fue desarrollado con fines **exclusivamente educativos**, con el objetivo de practicar:

- estructuras de datos (listas y diccionarios)
- algoritmos de ordenamiento
- manipulación de imágenes
